# Algorithms in C

A collection of fundamental algorithms and data structures implemented in C, providing educational examples and practical implementations for computer science students and developers.

## Overview

This repository contains clean, well-documented implementations of classic algorithms and data structures written in C. It's designed to serve as a learning resource and reference guide for understanding how fundamental CS concepts work at the implementation level.

## Included Algorithms & Data Structures

### Sorting Algorithms
- **BubbleSort** - Simple comparison-based sorting algorithm
- **Sort** - Additional sorting implementations

### Searching Algorithms
- **BinarySearch** - Efficient search algorithm for sorted data

### Data Structures
- **LinkedList** - Singly linked list implementation with fundamental operations
- **Heap** - Binary heap implementation for priority queue operations

### Utilities
- Helper functions and utilities for common operations
